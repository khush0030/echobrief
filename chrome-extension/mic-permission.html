Microphone Setup

EchoBrief needs microphone access to record your voice during meetings. Without it, only other participants' audio is captured.

How it works:
1. Click the button below
2. Chrome will ask to allow microphone access
3. Important: Click "Allow" (not dismiss/close the dialog)
4. If you see "Always allow," select that for uninterrupted recording
5. Go back to your meeting tab and start recording
If permission keeps failing:
Go to chrome://settings/content/microphone and make sure
EchoBrief is not in the "Blocked" list. Remove it if it is, then try again.
